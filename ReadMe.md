# 🏭 Factory Break ⛓️‍💥

_All commands should be run from the repository's root directory._

## 📦 Setup
1. Install repository's Python version.
    ```shell
    pyenv install
    ```
1. Install dependencies.
    ```shell
    pip install -r requirements.txt
    ```

## 👟 Running
1. Run the `main.py` file.
    ```shell
    python3 main.py
    ```
